# WordPress Theme Project

This project is a custom WordPress theme designed for a website that will feature both a blogs and an events.

## Project Goal:
To transform the static design files located within this folder into a fully functional WordPress theme. All content and elements must be dynamically managed through the WordPress Content Management System (CMS).

## Features:

### 1. Blog
- Standard blog functionality, with posts managed via WordPress.

### 2. Events
- Dedicated section for events, managed via a custom post type in WordPress.
- Each event will have a "Start Date" and an "End Date".
- Events will be categorized using a dedicated "Event Category", separate from standard blog categories.

## Deliverables:
A functional WordPress website can be delivered in one of two ways:

1.  **Zipped Project Folder:** Includes:
    *   The complete WordPress theme files.
    *   An SQL database dump.
    *   Login details for a test WordPress installation (admin user).
    *   A clear, concise method to self-host the project.

2.  **Deployed Instance Access:** Includes:
    *   A URL to the deployed project.
    *   Login details for a test WordPress installation (admin user).

## Requirements for Development:
- WordPress installation.
- PHP knowledge.
- Familiarity with WordPress theme development best practices.
- Knowledge of HTML, CSS, JavaScript.
